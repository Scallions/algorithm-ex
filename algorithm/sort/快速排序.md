# 快速排序


挑选一个中间元素，使用两个指向头和尾，使用中间元素来把区间划分为两个部分，然后再迭代对两个区间划分

```python

def qsort(nums, s, e):
    if s>=e:
        return
    m = nums[(s+e)/2]
    l = s
    r = e
    while l<=r:
        while nums[l]<m:
            l += 1
        while nums[r]>m:
            r -= 1
        if l<=r:
            nums[l], nums[r] = nums[r], nums[l]
            l += 1
            r -= 1
    qsort(nums, s, r)
    qsort(nums, l, e)
```
