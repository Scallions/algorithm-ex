# 双指针
双指针一般用于数组、链表、字符串等线性结构。双指针的复杂度一般为$O(n)$。

对于**数组**遍历搜索问题，需要固定第一个值搜索第二个值，若采用遍历则复杂度为$O(n^2)$，若**数组有序**采用二分搜第二个值，则为$O(n\log{n})$，若可以使用双指针则为$O(n)$。出了搜索外，双指针可以利用多个指针存储额外信息来解决一些其他问题。

滑动窗口、快慢指针是特殊的双指针。
